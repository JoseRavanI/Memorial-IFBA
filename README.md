# Memorial Virtual IFBA

Este projeto é um site estático que celebra a história, conquistas e memórias do Instituto Federal da Bahia (IFBA).

## Funcionalidades
- Navegação intuitiva entre páginas de história, entrevistas, arquivos e doações
- Linha do tempo com eventos marcantes do IFBA
- Galeria de fotos e depoimentos
- Página para doações e apoio institucional

## Estrutura
- index.html: Página inicial
- about.html: Sobre o IFBA e linha do tempo
- archive.html: Arquivo de fotos e eventos
- interviews.html: Entrevistas e depoimentos
- interview-detail.html: Detalhes de entrevistas
- donate.html: Informações para doação

## Tecnologias
- HTML, Tailwind CSS, Lucide Icons, AOS Animation

## Como usar
1. Clone ou baixe o repositório
2. Abra o arquivo index.html em seu navegador
